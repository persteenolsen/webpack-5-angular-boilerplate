Description


- This single page application was made in Angular 8 with a Webpack 5 configuration

Last Updated
- 09-01-2022

Main functionality
- Showing some of the power of Webpack
- Optimized production build by splitting the js and css files in seperate chunks
- Using Hot Reload / HMR by Webpack Dev Server
- Showing routing by Angular
- A custom 404 Error page. Click on the meny link Secret for a demo
- Support for some older browsers like IE 11 with Babel polyfills by cdn as well as polyfills.ts
- GET, POST, PUT and DELETE operations towards the test Web API
- Serving static assets like images, styles and fonts

Module Bundler
- Manuel configuration of the Webpack 5 from scratch

The single page application - The frontend
- Mainly Angular
- HTML
- PostCSS
- Bootstrap 4 by CDN for the responsive design

Security
- The only security is the use of HTTPS because it is a demo only

Hosting
- The application is hosted at a traditionel Webhotel

Text Editor
- Visual Studio Code
